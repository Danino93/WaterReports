## 📦 שלב 2: פתיחת הפרויקט

### א. חילוץ הקובץ
1. חלץ את הקובץ `WaterDamageReports.zip`
2. שמור את התיקייה במקום נגיש (למשל: `C:\Projects\` או `~/Projects/`)

### ב. פתיחה ב-Android Studio
1. פתח Android Studio
2. לחץ על `File` → `Open`
3. בחר את תיקיית `WaterDamageReports`
4. לחץ `OK`

### ג. יצירת local.properties
1. בתיקייה הראשית של הפרויקט, צור קובץ `local.properties`
2. הוסף שורה אחת עם הנתיב ל-SDK שלך:
   ```
   sdk.dir=C\:\\Users\\YourName\\AppData\\Local\\Android\\Sdk
   ```
   (ב-Mac/Linux: `sdk.dir=/Users/YourName/Library/Android/sdk`)

## ⏳ שלב 3: Gradle Sync

1. Android Studio יתחיל אוטומטית ב-**Gradle Sync**
2. תראה הודעה: "Gradle sync in progress..."
3. זה לוקח **5-10 דקות בפעם הראשונה** ⏰
4. המתן עד שתראה: "BUILD SUCCESSFUL"

### אם יש שגיאות:
1. לחץ על `File` → `Invalidate Caches / Restart...`
2. בחר `Invalidate and Restart`
3. המתן לסנכרון מחדש

## 🖥️ שלב 4: הרצה על אמולטור

### א. יצירת אמולטור
1. לחץ על `Tools` → `Device Manager`
2. לחץ `Create Device`
3. בחר:
   - **Category**: Phone
   - **Device**: Pixel 5 או חדש יותר
   - לחץ `Next`
4. בחר:
   - **System Image**: API 33 (Android 13.0)
   - אם לא מותקן, לחץ `Download` ליד התמונה
   - לחץ `Next`
5. שם לאמולטור: `Pixel_5_API_33`
6. לחץ `Finish`

### ב. הרצה
1. לחץ על הכפתור ▶️ (Run) בסרגל העליון
2. בחר את האמולטור שיצרת
3. המתן לטעינת האמולטור (דקה-שתיים)
4. האפליקציה תיפתח! 🎉

## 📱 שלב 5: הרצה על מכשיר אמיתי

### א. הפעלת Developer Options
1. **בטלפון Android שלך:**
   - פתח `הגדרות`
   - לך ל-`אודות הטלפון`
   - מצא `מספר גרסה` (או `Build Number`)
   - **לחץ 7 פעמים** על מספר הגרסה
   - תראה הודעה: "You are now a developer!"

### ב. הפעלת USB Debugging
1. חזור להגדרות
2. חפש `אפשרויות מפתח` (Developer Options)
3. הפעל את `USB Debugging`
4. אשר כשתשאל

### ג. חיבור למחשב
1. חבר את הטלפון למחשב עם כבל USB
2. בטלפון תופיע הודעה - אשר `Allow USB Debugging`
3. סמן `Always allow from this computer`

### ד. הרצה
1. ב-Android Studio, לחץ ▶️
2. בחר את הטלפון שלך מהרשימה
3. האפליקציה תותקן ותיפתח! 🚀

## 🛠️ פתרון בעיות נפוצות

### בעיה 1: "SDK location not found"
**פתרון:**
1. צור קובץ `local.properties` בתיקייה הראשית
2. הוסף:
   ```
   sdk.dir=C\:\\Users\\YourName\\AppData\\Local\\Android\\Sdk
   ```
3. Sync מחדש

### בעיה 2: "Gradle sync failed"
**פתרון:**
1. `File` → `Invalidate Caches`
2. `Invalidate and Restart`
3. המתן לסנכרון

### בעיה 3: "Device not found"
**פתרון:**
1. בטלפון: ודא ש-USB Debugging מופעל
2. נתק וחבר מחדש את הכבל
3. אשר את ההרשאה בטלפון

### בעיה 4: האפליקציה קורסת
**פתרון:**
1. `Build` → `Clean Project`
2. `Build` → `Rebuild Project`
3. הרץ מחדש

### בעיה 5: PDF לא נוצר
**פתרון:**
1. בדוק שיש הרשאות אחסון
2. הגדרות → אפליקציות → אשף האינסטלציה → הרשאות
3. אפשר `קבצים ומדיה`
