## 💡 **הנושאים הכי דחופים - מה אני ממליץ לתקן קודם:**

### **1. תיקיות/פרויקטים** 🔥🔥🔥
**למה זה חשוב:**
- ארגון טוב יותר
- אפשר לארגן עבודות לפי פרויקט
- קל למצוא עבודות

**מה צריך:**
```kotlin
@Entity(tableName = "folders")
data class Folder(
    @PrimaryKey(autoGenerate = true)
    val id: Long = 0,
    val name: String,
    val icon: String,
    val color: String,
    val dateCreated: Long
)

// הוספה ל-Job:
data class Job(
    ...
    val folderId: Long? = null // עבודה יכולה להיות בתיקייה
)
```

### **2. Excel מרוכז** 🔥🔥
**למה זה חשוב:**
- ללקוחות גדולים שרוצים לנתח נתונים
- ייצוא קל ל-Excel

**מה צריך:**
- Apache POI library
- פונקציה שיוצרת Excel מכל העבודות

### **3. CRM - אנשי קשר** 🔥🔥
**למה זה חשוב:**
- שמירת פרטי לקוחות
- קישור לקוח למספר עבודות

**מה צריך:**
```kotlin
@Entity(tableName = "contacts")
data class Contact(
    @PrimaryKey(autoGenerate = true)
    val id: Long = 0,
    val type: ContactType, // CUSTOMER, CONTRACTOR
    val firstName: String,
    val lastName: String,
    val phone: String,
    val email: String,
    val company: String?,
    val address: String?,
    val notes: String?
)
```
