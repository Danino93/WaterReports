## 📦 התקנה

### שלב 1: פתיחת הפרויקט

1. פתח **Android Studio**
2. לחץ על `File` → `Open`
3. בחר את התיקייה `WaterDamageReports`
4. לחץ `OK`

### שלב 2: Gradle Sync

1. Android Studio יתחיל אוטומטית ב-Gradle Sync
2. המתן עד לסיום ההורדה (5-10 דקות בפעם הראשונה)
3. אם יש שגיאות, לחץ על `Sync Now`

### שלב 3: הרצה על אמולטור

1. לחץ על `Tools` → `Device Manager`
2. צור אמולטור חדש:
   - Device: Pixel 5 או חדש יותר
   - API Level: 33 (Android 13)
3. לחץ על כפתור ▶️ (Run)
4. בחר את האמולטור
5. המתן לטעינה

### שלב 4: הרצה על מכשיר פיזי

1. הפעל **Developer Options** בטלפון:
   - הגדרות → אודות → לחץ 7 פעמים על "מספר גרסה"
2. הפעל **USB Debugging**:
   - הגדרות → אפשרויות מפתח → USB Debugging
3. חבר את הטלפון למחשב
4. לחץ ▶️ ובחר את הטלפון שלך

## 🔧 פתרון בעיות

### בעיה: Gradle Sync נכשל
**פתרון:**
1. `File` → `Invalidate Caches` → `Invalidate and Restart`
2. נסה שוב

### בעיה: האפליקציה קורסת
**פתרון:**
1. `Build` → `Clean Project`
2. `Build` → `Rebuild Project`
3. הרץ מחדש

### בעיה: PDF לא נוצר
**פתרון:**
1. בדוק הרשאות אחסון
2. בדוק שיש מקום פנוי
3. נסה להפחית איכות תמונות
